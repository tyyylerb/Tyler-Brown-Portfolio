my projects

The greatest way to learn something is by practicing and applying the skills you acquire. As I pursure both my college education as well as my own self-guided learning, I have put together multiple projects that show proof of my skills and reinforce essential software engineering principles while providing a mentally envigorating coding experience.

project 1: Playground

Playground is a work in progress project intended to help bring artists and creatives together in a collaborative space. Using basic blog functionality, the website will allow for users to create posts, leave comments on other users' posts, and sort created posts by user-assigned categories. An admin tab is also available for creating and removing posts, comments, and categories. This project uses Python and Django for the backend, a SQLite database, and HTML + CSS for the frontend formatting. (WIP, demonstration video)

project 2: Shortcuts

Shortcuts is a locally deployed tool that acts as a multi-purpose application. With a variety of buttons available, certain tasks can be automated, such as opening links. This project is built with Python using a Tkinter GUI. (WIP, demonstration video)

project 3: Responsive Web Design Project

This simple front end is the final product of my Responsive Web Design with HTML & CSS class. In this project, I learned how to use HTML, CSS, and certain elements of JavaScript to build a webpage for a hypothetical company that has all the fundamental functionalities of a modern website. (Demonstration video)
