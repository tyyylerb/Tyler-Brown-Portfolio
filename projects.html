my projects

The greatest way to learn something is by practicing and applying the skills you acquire. As I pursue both my college education as well as my own self-guided learning, I have put together multiple projects that show proof of my skills and reinforce essential software engineering principles while providing a mentally invigorating coding experience.

01

Playground

Playground is a work-in-progress project intended to help bring artists and creatives together in a collaborative space. Using basic blog functionality, the website allows users to create posts, leave comments on other users' posts, and sort posts by user-assigned categories. An admin tab is available for creating and removing posts, comments, and categories.

Built with Python, Django, SQLite, and HTML/CSS.

02

Shortcuts

Shortcuts is a locally deployed tool that acts as a multi-purpose application. With a variety of buttons available, certain tasks can be automated, such as opening links.

Built with Python using a Tkinter GUI.

04

Study Timer Website

While this project provides another example of working with a competent frontend stack, the core of it is much more personal. My girlfriend is majoring in Biology with a concentration in Cellular and Molecular Biology, and asked me to build her a simple website with a stopwatch for tracking her study progress.

Custom built with HTML, CSS, and JavaScript as a gift, and as evidence of my frontend design skills.
